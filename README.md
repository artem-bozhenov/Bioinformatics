# Описание проекта:
Этот проект содержит набор реализаций базовых структур данных на языке Python. Цель — предоставить готовые модули для работы с основными структурами данных, а так же некоторые функции для работы со строками и массивами в целом.
# Состав проекта
В проекте представлены следующие файлы:

- binary_trees.py — класс для реализации бинарных деревьев.
- graphs.py - класс для реализации графов
- doubly_linked_list.py — класс для реализации двусвязного списка.
- linked_list.py — класс для реализации связного списка.
- queue_using_linked_list.py — класс для реализации очереди на основе связного списка.
- stacks_implementation_linked_list.py — класс для реализации стека на основе связного списка.
- stacks_implementation_array.py — класс для реализации стека на основе массива.
- search_element_from_unsorted_array — функция для поиска элемента в несортированном массиве.
- sorted_array_insert.py — функция для вставки элемента в отсортированный массив.
- naive_substring_comparison.py - функция наивного поиска подстроки в строке.
