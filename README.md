Описание проекта:
Этот проект содержит набор реализаций базовых структур данных на языке Python. Цель — предоставить готовые модули для работы с основными структурами данных
Состав проекта
В проекте представлены следующие файлы:

BinTrees.py — реализация бинарных деревьев.
DoublyLinkedList.py — реализация двусвязного списка.
LinkedList.py — базовая реализация связного списка.
Queue_LinkedList.py — реализация очереди на основе связного списка.
Stacks_LinkedList.py — реализация стека на основе связного списка.
Stacks_array.py — реализация стека на основе массива.
search_element_from_unsorted_array — модуль для поиска элемента в несортированном массиве.
sorted_array_insert.py — модуль для вставки элемента в отсортированный массив.
