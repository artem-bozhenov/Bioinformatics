# Описание проекта:
Этот проект содержит набор реализаций базовых структур данных на языке Python. Цель — предоставить готовые модули для работы с основными структурами данных, а так же некоторые функции для работы со строками и массивами в целом.
# Состав проекта
В проекте представлены следующие файлы:

- BinTrees.py — класс для реализации бинарных деревьев.
- - 
- Graphs.py - класс для реализации графов
- DoublyLinkedList.py — класс для реализации двусвязного списка.
- LinkedList.py — класс для реализации связного списка.
- Queue_LinkedList.py — класс для реализации очереди на основе связного списка.
- Stacks_LinkedList.py — класс для реализации стека на основе связного списка.
- Stacks_array.py — класс для реализации стека на основе массива.
- search_element_from_unsorted_array — функция для поиска элемента в несортированном массиве.
- sorted_array_insert.py — функция для вставки элемента в отсортированный массив.
- Naive_SubStrCmp.py - функция наивного поиска подстроки в строке
- rabin_carp.py - функция поиска подстроки в строке методом Рабина-Карпа
